## Run Manually

clone the repo

```
cd build/
```

then open up index.html with ```--allow-file-access-from-files``` argument.

For Chromium run:

```
chromium index.html --allow-file-access-from-files 
```

Similarly, For Firefox:

```
firefox index.html --allow-file-access-from-files 
```
